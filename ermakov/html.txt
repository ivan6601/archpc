2. назначение параметров Функции (параметров можети и не быть)
3. возврат значений (например с помощью return)
4. вызов функции в нужный момент myfunc (аргументы)
function myfunc (параметры) {

параметры функции - это локальные переменные внутри функции
myfunc(a,b){
return a+b }
